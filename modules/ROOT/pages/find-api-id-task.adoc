= Determining API Instance ID on API Manager
ifndef::env-site,env-github[]
include::_attributes.adoc[]
endif::[]

The API Instance ID is displayed in API Manager on the API Instance dashboard. This unique system value is generated, and cannot be changed.

CAUTION: The API Instance ID is not available for Mule 3 autodiscovery use in API Manager.

. On the API administration dashboard, expand the API version name to display the instances.
+
image::apis-api-manager-dashboard.png[]
+
. Click an instance name.
+
The API Instance ID is displayed below the API Instance name. 
+
image::display-api-id-api-manager.png[]
