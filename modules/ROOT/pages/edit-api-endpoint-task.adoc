= Editing an API Instance Configuration Settings
ifndef::env-site,env-github[]
include::_attributes.adoc[]
endif::[]

After you create an API instance in API Manager, you can edit the instance's configuration settings.

For more information about the API instance configuration fields, see the relevant table for runtime from xref:api-configuration-reference.adoc[].

To edit an API instance's configuration settings:

. Navigate to *Anypoint Platform* > *API Manager*.
. In *API Administration*, click the name of the API instance whose configuration settings you want to edit. 
. Click *Settings*.
. Edit the desired configuration fields.
. Click *Save*.
